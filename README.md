# 📌 TrackWise – Data Retrieval & Sorting Dashboard

## 📖 Description

TrackWise is a full-stack web application designed to efficiently manage, retrieve, and sort data through an interactive dashboard. It allows users to store and view records dynamically using a clean and user-friendly interface.

This project demonstrates core full-stack concepts including frontend design, backend APIs, and database integration.

---

## 🚀 Features

* 📊 Interactive Dashboard
* 🔍 Data Retrieval from Database
* 🔄 Sorting Functionality (Name, Department, etc.)
* 📝 Add and Manage Records
* ⚡ Real-time Data Updates
* 🌐 Simple and Responsive UI

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Faheem-cloud/TrackWise.git
cd TrackWise
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start Server

```bash
node server.js
```

### 4. Open in Browser

```
http://localhost:3000
```

---

## 📊 How It Works

1. User opens the application
2. Enters data through registration form
3. Data is stored in MySQL database
4. Dashboard fetches and displays records
5. User can sort data dynamically
